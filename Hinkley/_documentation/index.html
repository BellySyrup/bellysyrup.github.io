12 Column Grid

This is a full responsive 12 Column Grid.
If you want a margin between the columns, add the class "gutters" to the "row" <div> like this:
<div class="row gutters"></div>
If you want no margin, remove the class "gutters". 

<!-- START SECTION -->
<section id="section-id" class="section">
	<!-- START CONTAINER -->
    <div class="row gutters">
        <div class="col span_6">
			{{ 6/12 COLUMN PAGE CONTENT }}
        </div>
        <div class="col span_6">
			{{ 6/12 COLUMN PAGE CONTENT }}
        </div>
	</div><!-- END CONTAINER -->
</section><!-- END SECTION -->

Theme Javascript

There are two theme javascript files. The first contains the script for the navigation, header, work-preview, blog-preview and shortcodes. You do not need to edit these file. If you want to change settings, add a slider, a parallax background or something else, you must edit the "theme.settings.js" file.

1. theme.script.js - Theme script
2. theme.settings.js - Do all settings here

Add a Parallax Background

Html

To give a container a parallax background add the class="paralax-content" and a individual class name to define the background image
class="paralax-content bubble-background". 

<div class="paralax-content bubble-background">
    Your Content
</div>

CSS - stylesheet.css

Now add your new class and define the background image. 

.bubble-background {
    background-image: url('../path/bubble-background.jpg');
}

Javascript - theme.settings.js

If you want different parallax sections with different speeds, copy the javascript code and replace the class ".paralax-content" with the class of your section. 

$('.paralax-content').parallax({
    speed: 0.25 /* speed of the parallax effect */
});
